## 🌍 Overview

**Domain Harvester** is a web-based intelligence platform that:

- 🌐 Collects domains from multiple sources  
- 🧠 Classifies them by industry  
- 🔍 Validates DNS records  
- 💾 Stores clean structured data  
- 📤 Exports in multiple formats  

---

## ⚙️ Architecture

### Frontend
- Vanilla HTML / CSS / JS  
- Single-page dashboard  
- Theme system  

### Backend
- FastAPI  
- APScheduler  
- Modular scraping engine
